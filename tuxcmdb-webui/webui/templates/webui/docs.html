Overview

TuxCMDB uses a FastAPI backend. Main endpoint groups:

  • Assets and assignments
  • Attributes and fetch methods
  • Operating systems and aliases
  • Datatypes
  • Agent bootstrap and report

Health checks:

Examples

List assets:

curl -u USER:PASSWORD {{ api_url }}/v1/assets

Assign value to asset:

curl -u USER:PASSWORD -X POST \
  -H 'Content-Type: application/json' \
  -d '{"memory_gb":"32Gi"}' \
  {{ api_url }}/v1/assets/7/attributes

Remove assignment value:

curl -u USER:PASSWORD -X DELETE \
  '{{ api_url }}/v1/assets/7/attributes/memory_gb?value=32Gi'
